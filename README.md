# Squad Goals - Dream Cricket Team

Welcome to **Thunder Strikers** - a fan-made dream team web app made with HTML, CSS, and JS!

## Features
- Team Name, Logo, and Slogan
- Player Grid with images and roles
- Modal with bonus stats:
  - Strike Rate
  - Matches Played
  - Jersey Number
  - Best Performance
